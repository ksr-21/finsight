# Finsight PWA Integration & APK Generation Guide

This guide provides instructions for integrating the generated `manifest.json` and service worker into your React/Vite application and preparing it for APK generation.

## 1. PWA Manifest Integration

The `manifest.json` file is generated by Vite during the build process based on the configuration in `vite.config.ts`.

### HTML Integration (`index.html`)
Ensure the following tags are present in your `<head>`:
```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#4F46E5">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<meta name="apple-mobile-web-app-title" content="Finsight">
```

## 2. Service Worker Registration

The project uses `vite-plugin-pwa` with `registerType: 'autoUpdate'`, which handles service worker registration automatically. A manual `serviceWorkerRegistration.ts` is also provided if you prefer a custom approach.

## 3. APK Generation via PWABuilder

To generate a production-ready Android APK:

1. **Deploy your app:** Host your application on a HTTPS-enabled server.
2. **Visit PWABuilder:** Go to [pwabuilder.com](https://www.pwabuilder.com).
3. **Analyze URL:** Enter your deployed URL.
4. **Package for Android:**
   - Click "Package for Stores".
   - Select "Android".
   - **Package ID:** Use `com.finsight.app`.
   - **App Name:** Finsight.
5. **Download:** Download the generated ZIP file.

## 4. Key PWA Features
- **Offline First:** Assets are cached for offline access.
- **Shortcuts:** Quick access to Dashboard, Transactions, Insights, and Budgets.
- **Maskable Icons:** Adaptive icon support for Android.
- **Standalone Mode:** Native app-like experience.
